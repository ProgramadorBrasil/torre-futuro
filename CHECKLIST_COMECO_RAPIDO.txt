================================================================================
  TROUBLESHOOTING RAPIDO
================================================================================

PROBLEMA: Unity Hub nao abre
SOLUCAO:
  1. Reinstale Unity Hub
  2. Execute como Administrador
  3. Verifique firewall/antivirus

────────────────────────────────────────────────────────────────────

PROBLEMA: Projeto nao abre (erro de versao)
SOLUCAO:
  1. Verifique ProjectVersion.txt
  2. Instale versao correta do Unity
  3. Ou tente versao superior (fara upgrade)
  4. Delete pasta Library e tente novamente

────────────────────────────────────────────────────────────────────

PROBLEMA: Erros vermelhos no Console
SOLUCAO:
  1. Leia o PRIMEIRO erro (ignore resto)
  2. Duplo-clique no erro para ver codigo
  3. Google o erro especifico
  4. Verifique se scripts estao completos
  5. Veja TROUBLESHOOTING_RAPIDO.txt para detalhes

────────────────────────────────────────────────────────────────────

PROBLEMA: Controles nao funcionam
SOLUCAO:
  1. Clique dentro do Game View (dar foco)
  2. Verifique SpaceshipController esta enabled
  3. Check InputManager: Edit > Project Settings > Input
  4. Veja Console por erros do controller

────────────────────────────────────────────────────────────────────

PROBLEMA: Sem audio
SOLUCAO:
  1. Verifique volume do computador
  2. Main Camera deve ter Audio Listener
  3. AudioSource components devem estar enabled
  4. Audio clips devem estar assignados
  5. Volume > 0 no AudioSource

────────────────────────────────────────────────────────────────────

PROBLEMA: FPS muito baixo
SOLUCAO:
  1. Edit > Project Settings > Quality > Level: Low
  2. Desabilite VSync
  3. Reduza resolucao no Game View
  4. Feche outros programas
  5. Veja Unity Profiler para bottlenecks

────────────────────────────────────────────────────────────────────

PROBLEMA: Tela preta ao apertar Play
SOLUCAO:
  1. Verifique Main Camera existe no Hierarchy
  2. Camera position: (0, 0, -10) aproximadamente
  3. GameManager deve estar na cena
  4. Verifique Console por erros de inicializacao

────────────────────────────────────────────────────────────────────

PROBLEMA: UI nao aparece
SOLUCAO:
  1. Verifique Canvas existe e esta enabled
  2. Canvas Render Mode: Screen Space - Overlay
  3. Elementos UI sao filhos do Canvas
  4. Layer do Canvas e UI

────────────────────────────────────────────────────────────────────

PROBLEMA: NullReferenceException
SOLUCAO:
  1. Duplo-clique erro para ver linha
  2. Verifique variaveis assignadas no Inspector
  3. Adicione null checks no codigo
  4. Inicialize variaveis no Start() ou Awake()

────────────────────────────────────────────────────────────────────

PROBLEMA: Mudancas nao salvam
SOLUCAO:
  1. NAO faca mudancas durante Play Mode!
  2. Pare o jogo primeiro (Stop)
  3. Faca mudancas
  4. Ctrl+S para salvar cena
  5. File > Save Project
