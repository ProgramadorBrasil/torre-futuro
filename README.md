# Torre Futuro

Jogo de torre desenvolvido em Unity.

## Estrutura do Projeto

- `Assets/` - Arquivos do projeto Unity (cenas, scripts, recursos, etc.)
- `ProjectSettings/` - Configurações do projeto Unity
- `Packages/` - Dependências do projeto

## Como Executar

1. Abra o projeto com Unity
2. Abra a cena principal em `Assets/Scenes/`
3. Clique em Play para testar

## Contribuindo

Siga os padrões de código estabelecidos no projeto.
