SE ALGO DER ERRADO

❌ "Safe Mode aparece"
   SOLUÇÃO:
   1. Feche Unity completamente
   2. Abra File Explorer
   3. Navegue para D:\games\torre futuro
   4. Delete pasta "Library"
   5. Reabra Unity - vai recompilar
   6. Espere compilação terminar

❌ "Console mostra erros vermelhos"
   SOLUÇÃO:
   1. Clique no erro para ver detalhes
   2. Se for "DOTween not found" = ignore (fallback em uso)
   3. Se for "Burst not found" = ignore (fallback em uso)
   4. Se for outro erro: salve screenshot e reporte

❌ "Texturas aparecem magenta"
   SOLUÇÃO:
   1. Assets > Reimport All
   2. Aguarde importação
   3. Tente novamente

❌ "Jogo não inicia"
   SOLUÇÃO:
   1. Clique na cena MainMenu
   2. Veja console para erros
   3. Se houver NullReferenceException: verifique assigns nos Inspectors

❌ "Baixa performance"
   SOLUÇÃO:
   1. Edit > Project Settings > Quality
   2. Mude para "Fast"
   3. Feche outros programas
   4. Tente novamente
