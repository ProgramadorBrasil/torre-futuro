TIPO: Null reference bug
SEVERIDADE: CRITICAL

PROBLEMA:
  private List<RewardData> pendingRewards = new Queue<RewardData>().ToArray() as List<RewardData>;

Explicacao:
  1. new Queue<RewardData>() - cria fila vazia
  2. .ToArray() - converte para RewardData[] (array)
  3. as List<RewardData> - tenta fazer CAST de array para List
  4. RESULTADO: Retorna NULL (nao pode fazer cast de array para List)

ANTES (BUGADO):
  private List<RewardData> pendingRewards = new Queue<RewardData>().ToArray() as List<RewardData>;
  // ^ Este codigo deixa pendingRewards = null !

DEPOIS (CORRIGIDO):
  private List<RewardData> pendingRewards = new List<RewardData>();
  // ^ Agora inicializa List vazia corretamente
